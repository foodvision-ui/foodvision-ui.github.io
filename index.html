How It Works

One loop. Three simple moments.

This is the part the reference site does especially well: the flow is broken into short steps users can understand in seconds. Your app has a strong version of that too.

1

Snap or upload

Start from a food photo, fridge photo, or a craving. The user does not need to build a perfect profile before the product becomes useful.

2

Generate with context

The planner combines image understanding, current profile data, and recent meal history so recommendations feel more intentional than one-off prompt completions.

3

Learn for next time

After saving the meal, the system can evolve the profile and show which preferences were inferred. The next session starts smarter than the last one.
